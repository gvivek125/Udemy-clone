Udemy Clone – Responsive Website
---------------------------------

This project is a responsive front-end clone of the Udemy website, created using HTML5 and CSS3. The main goal of this project is to practice real-world UI design, layout structuring, and responsive web design techniques.

The website adapts smoothly to different screen sizes including desktop, tablet, and mobile, ensuring a consistent user experience across devices.

Features
---------

* ✅ Fully responsive design
* ✅ Udemy-like homepage layout
* ✅ Course cards with images, title, and pricing
* ✅ Navigation bar & footer section
* ✅ Clean and structured HTML
* ✅ CSS Flexbox & Media Queries used

Technologies Used
-----------------

* HTML5
* CSS3
* Flexbox
* Media Queries

Purpose of the Project
-------------------------

This project was developed to:
-----------------------------

* Improve front-end development skills
* Understand responsive UI design
* Practice cloning real-world websites
* Build a strong portfolio project
